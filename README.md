# greeting-identifier-module
This is a machine learning model created from Teachable Machines for _demonstration_ purposes.

**To get started building your own Machine Learning Model (MLM)**: https://teachablemachine.withgoogle.com/

The **easiest** process to deploy the Greeting Identifier module is to copy and paste the following URL into the broswer:
https://teachablemachine.withgoogle.com/models/6eSL5RTKA/

You can also download the the "Greeting Identifier.tm" Model file and use *Open project from file* on *Teachable Machine*
